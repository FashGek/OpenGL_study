## 图形渲染管线（Graphics Pipeline）

指的是一堆原始图形数据途径一个输送管道，期间经过各种变化处理，最终出现在屏幕的过程

可以被划分为两个主要部分：

1. 3D坐标 -> 2D坐标
2. 2D坐标 -> 有颜色的像素
